# Handoff: gui-static-mock

**Статус:** waiting hardware  
**Канон:** `sketch_jul19a/sketch_jul19a.ino`  
**Дата:** 2026-07-19  
**PlatformIO `src/main.cpp`:** не синхронизировать

## Цель

Набросать **условный статический GUI** 320×240 по макету Scale Master (`New scale.svg` / PNG): три шкалы (скорость, газ, заряд), цвета и грубая геометрия. Без датчиков, без анимации стрелок в loop, без меню.

## Контракт

- Дисплей ILI9341, **Arduino_GFX**, явная шина Nano (не `create_default` — библиотека пропатчена под Mega).
- Пины TFT (провизорно, как сток Nano в эталоне GFX; сменить после разводки):
  - SPI: MOSI **11**, SCK **13**, MISO **12**
  - DC **8**, CS **9**, RST **7**
- Rotation **1** → 320×240 landscape.
- Фон `#000000`; спидометр дуги `#808080` / `#ffffff`; зоны малых шкал `#00ff00` / `#ffff00` / `#ff0000`; стрелки `#ffff00`.
- Центры/радиусы — масштаб с SVG 397×280 → 320×240 (см. AGENTS.md).
- Цифровые TORQUE / U / км·ч под шкалами — **заглушки** (текст), точная вёрстка позже.
- `loop()` пустой или только `yield`; без `delay`.

## Acceptance

- [ ] Прошивка заливается на Nano
- [ ] На ILI9341 видны 3 шкалы, похожие на макет (допуск: «условно»)
- [ ] Termoregular / библиотека Mega не ломаются (явные пины в sketch)

## Out of scope

- Датчики, кулон, поворотники
- Динамические стрелки / dirty-update
- Точная копия всех меток SVG
- Смена пинов CS/DC/RST без обновления AGENTS
