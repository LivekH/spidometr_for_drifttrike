# Handoff: gui-from-draw-master

**Статус:** waiting hardware  
**Канон:** `sketch_jul19a/sketch_jul19a.ino`  
**Дата:** 2026-07-24  
**PlatformIO `src/main.cpp`:** не синхронизировать

## Цель

Заменить условный mock GUI на статическую отрисовку из **GUI Draw Master** (экспорт Arduino_GFX), 320×240, rotation 1.

## Контракт

- Библиотека: Arduino_GFX (`lib/Arduino_GFX-master` / явные пины Nano).
- Объект: `Arduino_GFX *gfx` (как в экспорте `gfx->`).
- Пины TFT: DC **9**, CS **10**; SPI MOSI **11**, SCK **13**; RST `GFX_NOT_DEFINED` (пока).
- `drawGui()` один раз в `setup()` после `gfx->begin()`.
- Без датчиков, без стрелок в `loop`, окно под спидометром пустое (рамка есть).
- Углы дуг — из утверждённого экспорта (газ: 20→55→90→125; вольтметр: 235→340; спидометр шкала 150→30; кольцо 0→360).

## Acceptance

- [ ] Компилируется / заливается на Nano
- [ ] На ILI9341 2.4" видны спидометр + вольтметр + газ + рамка окна
- [ ] Секторы газа/вольтметра без «длинных» дуг на весь круг

## Out of scope

- Динамические стрелки, цифры в окне, датчики, поворотники
- Смена пинов TFT без обновления AGENTS
